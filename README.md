🚀 Universal Video Downloader

Universal Video Downloader — это инструмент на Python для удобного скачивания видео и аудио. Программа поддерживает работу с сотнями площадок и предоставляет гибкие настройки через интерактивные команды.

✨ Возможности

🔍 Умный поиск: Просто введите название видео, и скрипт найдет его на YouTube.

📦 Пакетная работа: Поддержка ввода нескольких ссылок через запятую или пробел.

🎵 Режим аудио: Быстрая конвертация видео в MP3 (при наличии FFmpeg).

📉 Выбор качества: Настройка разрешения от минимального до максимально доступного.

💾 Логирование: Автоматическое ведение отчетов о загрузках.

📁 Структура проекта

При первом запуске программа автоматически создаст следующие папки:

downloads/ — сюда будут сохраняться все ваши видео и аудио файлы.

logs/ — здесь хранятся текстовые отчеты о работе программы (помогают понять, что пошло не так, если загрузка прервалась).

🚀 Инструкция по установке

1. Подготовка

Убедитесь, что у вас установлен Python.

2. Установка зависимостей

Выполните команду в терминале:

    pip install yt-dlp


3. Установка FFmpeg (Рекомендуется)

Для склейки видео в высоком качестве (1080p+) и конвертации в MP3 скачайте FFmpeg и добавьте его в переменную окружения PATH.

💻 Использование

Интерактивный режим

Запустите скрипт:

    python video_downloader.py


Доступные команды внутри программы:

:quality 1080 — ограничить качество видео.

:audio — переключить режим (видео / только звук).

:stats — показать статистику текущей сессии.

:help — список всех команд.

выход — завершение работы.

Консольный режим (CLI)

Для быстрой загрузки без входа в меню:

    python video_downloader.py "ссылка_1" "ссылка_2" -q 1080 --audio
